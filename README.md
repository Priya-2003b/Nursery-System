# Nursery Management System

A web-based Nursery Management System designed to manage plant inventory, customer orders, sales, and daily operations efficiently. Built using PHP, MySQL, HTML, CSS, and JavaScript.

## Features
- User login and authentication
- Plant and product inventory management
- Stock level monitoring
- Customer management
- Sales and order tracking
- Reports for sales and inventory
- Responsive user interface

## Technologies Used
- Backend: PHP
- Database: MySQL
- Frontend: HTML, CSS, JavaScript
- Server: Apache (XAMPP/WAMP)
